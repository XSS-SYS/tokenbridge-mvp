API 文档

TokenBridge 提供与 OpenAI API 完全兼容的接口，只需替换 base URL 即可无缝切换。支持 Python / Node.js / Go / Java / PHP / Rust / Ruby / Swift 等主流语言。

概述

TokenBridge 是一个高性能的 AI API 中转服务，通过 OpenAI 兼容接口提供 DeepSeek V4 Flash 模型的访问。只需修改 base URL，无需改动代码。

🔗 Base URL：https://tb-api.top/v1
✅ API Key 格式：tb_xxxxx... — 支付完成后发送到您的邮箱

快速开始

  1. 访问 首页 → 在套餐区选择套餐 → 支付获取 API Key
  2. 将 API Key 作为 Bearer Token 传入请求头
  3. 调用 /v1/chat/completions 端点，指定模型 ID

或直接用公共测试 Key 体验（无需付款）：

bash
curl https://tb-api.top/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer tb_test_free" \
  -d '{"model":"deepseek-chat","messages":[{"role":"user","content":"Hello!"}]}'
⚠️ 公共测试 Key — 共享额度 1M Tokens，每 IP 每分钟限 5 次。仅用于快速体验，生产环境请使用专属 Key。
→ 获取专属 Key 用于生产

认证方式

使用 Bearer Token 认证，将 API Key 放入 HTTP 请求头：

http
Authorization: Bearer tb_your_api_key_here

所有请求都必须包含 Authorization 头，否则返回 401 Unauthorized。

聊天补全

与 OpenAI chat/completions 完全兼容，支持所有标准参数。

POST/v1/chat/completions

创建聊天补全请求

请求参数

参数类型必需说明
modelstring是模型 ID，如 deepseek-chat、deepseek-v4-flash
messagesarray是消息列表，支持 role: system / user / assistant
streamboolean否是否流式输出，默认 false
temperaturenumber否生成温度 (0–2)，默认 1.0
max_tokensnumber否最大输出 tokens
top_pnumber否核采样参数 (0–1)，默认 1.0
frequency_penaltynumber否频率惩罚 (-2 到 2)，默认 0
presence_penaltynumber否存在惩罚 (-2 到 2)，默认 0
stopstring / array否停止词，最多 4 个

流式输出 (SSE)

设置 "stream": true 启用 SSE 流式输出，实时获取模型生成内容。响应格式（每行以 data: 开头）：

text
data: {"choices":[{"delta":{"content":"你好"},"index":0}]}
data: {"choices":[{"delta":{"content":"！"},"index":0}]}
...
data: [DONE]
💡 使用建议：流式输出适合聊天类应用。非流式适合批量处理场景。

python
import httpx, json
async with httpx.AsyncClient() as c:
    async with c.stream("POST", "https://tb-api.top/v1/chat/completions",
        headers={"Authorization": "Bearer tb_your_api_key"},
        json={"model": "deepseek-chat", "messages": [{"role": "user", "content": "你好"}], "stream": True}
    ) as r:
        async for line in r.aiter_lines():
            if line.startswith("data: ") and line != "data: [DONE]":
                c = json.loads(line[6:])["choices"][0]["delta"].get("content","")
                print(c, end="")

常见问题

Q: TokenBridge 和直接用 OpenAI / DeepSeek 有什么区别？

完全兼容 OpenAI 接口，只需改 base_url。国内直接访问，无需代理。一个 Key 调用所有 DeepSeek 模型。

Q: 套餐 tokens 怎么算法？

输入 + 输出 tokens 均计入配额。月付套餐每月重置。充值包永久有效。

Q: 支持流式（Streaming）吗？

支持。设置 "stream": true 即可通过 SSE 获取逐字输出。

Q: 可以退款吗？

购买后 7 天内未使用的配额可申请退款。已使用部分按比例扣减。

Q: 用量从哪里看？

在 首页用量区域 输入 Key 即可查询剩余额度和使用记录。
