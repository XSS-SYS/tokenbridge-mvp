❓ 常见问题

快速上手 TokenBridge

如何接入？▼
只需将 base URL 替换为 https://tb-api.top/v1，然后使用你的 API Key 鉴权即可。

示例 (Python)：
from openai import OpenAI
client = OpenAI(api_key="tb_xxx", base_url="https://tb-api.top/v1")
response = client.chat.completions.create(model="deepseek-chat", messages=[{"role":"user","content":"你好"}])
支持哪些模型？▼
DeepSeek V4 Flash (deepseek-chat) 及后续更新的 DeepSeek 系列模型。
全部通过同一个端点 /v1/chat/completions 调用。详情见 API 文档。

Pro Flash $29/月含 100M Tokens，超出 $0.40/M。Team Flash $99/月含 500M Tokens，超出 $0.30/M。
Pro Pro $199/月含 100M Tokens（v4-pro 模型）。
