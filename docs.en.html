API Documentation

TokenBridge provides fully OpenAI-compatible API endpoints. Simply change your base URL to switch. Supports Python, Node.js, Go, Java, PHP, Rust, Ruby, Swift, and more.

Overview

TokenBridge is a high-performance AI API proxy service providing access to DeepSeek V4 Flash (and upcoming DeepSeek Pro) through an OpenAI-compatible endpoint. Simply swap your base_url — no code changes needed.

🔗 Base URL: https://tb-api.top/v1
✅ API Key Format: tb_xxxxx... — delivered to your email after purchase

Quick Start

  1. Visit the homepage → choose a plan → purchase to get your API Key
  2. Pass your API Key as a Bearer Token in the Authorization header
  3. Call the /v1/chat/completions endpoint with your chosen model ID

Or try it right now with our public test key (no purchase needed):

bash
curl https://tb-api.top/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer tb_test_free" \
  -d '{"model":"deepseek-chat","messages":[{"role":"user","content":"Hello!"}]}'
⚠️ Public Test Key — Shared quota of 1M tokens, rate-limited to 5 req/min per IP. For quick evaluation only.
→ Get your own API Key for production

Chat Completions

Fully compatible with OpenAI's chat/completions. All standard parameters supported.

POST/v1/chat/completions

Create a chat completion

Request Body Parameters

ParameterTypeRequiredDescription
modelstringYesModel ID, e.g. deepseek-chat, deepseek-v4-flash
messagesarrayYesMessage list with role: system / user / assistant
streambooleanNoEnable SSE streaming. Default false
temperaturenumberNoSampling temperature (0–2). Default 1.0
max_tokensnumberNoMaximum output tokens
top_pnumberNoNucleus sampling (0–1). Default 1.0
frequency_penaltynumberNoFrequency penalty (-2 to 2). Default 0
presence_penaltynumberNoPresence penalty (-2 to 2). Default 0
stopstring / arrayNoStop sequences (max 4)

Streaming (SSE)

Set "stream": true to enable SSE streaming. Each line starts with data: :

text
data: {"choices":[{"delta":{"content":"Hello"},"index":0}]}
data: {"choices":[{"delta":{"content":"!"},"index":0}]}
...
data: [DONE]

python
import httpx, json
async with httpx.AsyncClient() as c:
    async with c.stream("POST", "https://tb-api.top/v1/chat/completions",
        headers={"Authorization": "Bearer tb_your_api_key"},
        json={"model": "deepseek-chat", "messages": [{"role": "user", "content": "Hello"}], "stream": True}
    ) as r:
        async for line in r.aiter_lines():
            if line.startswith("data: ") and line != "data: [DONE]":
                c = json.loads(line[6:])["choices"][0]["delta"].get("content","")
                print(c, end="")

while(true) { const {done,value} = await reader.read(); if(done) break;
  for(const line of decoder.decode(value).split('\n')) {
    if(line.startsWith('data: ') && line!=='data: [DONE]') {
      const c = JSON.parse(line.slice(6)).choices?.[0]?.delta?.content;
      if(c) process.stdout.write(c);
    }
  }

FAQ

Q: How is TokenBridge different from using OpenAI / DeepSeek directly?

Fully OpenAI-compatible interface. Just change base_url. Accessible from China without a proxy. One key for DeepSeek models.

Q: How are tokens counted?

Both input and output tokens count toward your quota. Monthly plans reset each renewal.

Q: Do you support streaming?

Yes. Set "stream": true to receive SSE-formatted token-by-token output.

Q: Can I get a refund?

Unused quota within 7 days of purchase is refundable. Used portion is deducted proportionally.

Q: Where can I check my usage?

Enter your API Key in the usage section on the homepage.
