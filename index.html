❓ FAQ

Get started with TokenBridge in minutes.

How do I connect?▼
Just set your base URL to https://tb-api.top/v1 and authenticate with your API Key.

Python example:
from openai import OpenAI
client = OpenAI(api_key="tb_xxx", base_url="https://tb-api.top/v1")
response = client.chat.completions.create(model="deepseek-chat", messages=[{"role":"user","content":"Hello!"}])
Which models are supported?▼
DeepSeek V4 Flash (deepseek-chat) and upcoming DeepSeek models.
All through the same /v1/chat/completions endpoint. See API Docs.
How does pricing work?▼
Pro Flash $29/mo includes 100M Tokens, overage at $0.40/M. Team Flash $99/mo includes 500M Tokens, overage at $0.30/M.
Pro Pro $199/mo includes 100M Tokens (v4-pro model).
What payment methods do you accept?▼
₮ USDT (TRC-20). Plans start at $29/mo for Pro Flash. After payment, your API Key is sent to your email.
How do I get an API Key?▼
Your API Key is sent to your email automatically after payment. You can also look up existing keys by email on this page.
Admins can create keys manually in the Admin Dashboard.
